# **App Name**: Dose.

## Core Features:

- Disclaimer Modal: Display a legal disclaimer on the first app launch, requiring user agreement to proceed, with an 'I am a doctor and I agree' button.
- Real-time Search: Implement a search bar that filters prescription cards in real-time as the user types, searching by condition or drug name.
- Prescription Cards: Display clinical information and prescribing guidances inside clear, well-structured cards that present the critical clinical details.
- Data Storage and Retrieval: Store prescription data in local JSON files for offline access and quick retrieval.
- Progressive Web App (PWA) Support: Configure the application as a PWA, allowing users to install it on their devices for an app-like experience.
- Dark Mode Toggle: Include a toggle for switching between light and dark themes for comfortable viewing in different environments.

## Style Guidelines:

- Primary color: Royal Blue (#2563EB) for interactive elements and highlights.
- Light background: Ice White (#F8FAFC) for a clean, clinical feel.
- Dark background: Slate Dark (#0F172A) for dark mode.
- Text color (light mode): Charcoal Gray (#334155) for readability.
- Text color (dark mode): Soft White (#E2E8F0) for legibility on dark backgrounds.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, clinical appearance. Note: currently only Google Fonts are supported.
- Mobile-first design with a focus on simplicity and speed.  Prominent search bar, rounded cards.
- Use 'Lucide React' icons for a consistent and clean interface.
